## üî® Compilation

All CPP modules are written in **C++98** and follow strict compilation requirements:

```bash
# Standard compilation command used across all modules
c++ -Wall -Wextra -Werror -std=c++98 [source_files] -o [executable_name]
```

### Requirements:
- **Compiler:** c++ (clang++ or g++)
- **Standard:** C++98
- **Flags:** `-Wall -Wextra -Werror`
- **No external libraries** (except STL where specified)

Each module repository contains its own Makefile with appropriate compilation rules.
